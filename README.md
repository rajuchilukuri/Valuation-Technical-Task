## Tools Used
```
Java 
Apache Maven 
Selenium 
Cucumber
Junit
Webdriver manager
```

## Commands:

## To run with Chrome
```
mvn clean install
```
## To run with Chrome headless
```
mvn clean install -DheadLess=true
```
## To run with Firefox
```
mvn clean install -Dbrowser=firefox
```
## To run with Safari
```
mvn clean install -Dbrowser=safari
```
